# Rock Paper Scissors GUI with Python
Simple rock paper scissors game with GUI

## Features
- Simple buttons
- Submit and Reset button
- Toggleable buttons
- Updateable texts

## Dependencies
- Made in Python 3.9.5
- Random module
- Tkinter module (version 8.6)
